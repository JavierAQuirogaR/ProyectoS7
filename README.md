# ProyectoS7
Este proyecto permite revisar los conceptos recibidos en el uso del Streamlit y del Render para visualizar una página web utilizando Python y librerías gráficas.

## Propósito
Utilizar las características del **streamlit** y el **Render** para mostrar una página web elaborada en Python sin tener que configurar un servidor Web para tal fin. 
## DataSet
No se requiere un dataset pues se trata solo de visualizar textos o imágenes.
## Resultados
Se visualiza la página Web con el contenido descrito en el código Python
